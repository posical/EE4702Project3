startFrom()method 
public function startFrom(start:int):void

Set the clock to assume starting from a specific time. For example aClock.startFrom(1000) will assume 1000 milliseconds have elapsed for aClock and start counting from 1000 milliseconds. aClock.reset() should be used instead of aClock.startFrom(0) to avoid unnecessary overhead.

Parameters

start:int — The start time in milliseconds. (Total time assumed to have elapsed)
