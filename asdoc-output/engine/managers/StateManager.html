addActiveState()method
public function addActiveState(state:State):void

Add an active State. Add a State and set it to be active. Only the active State will be added to stage. Active stage will be initialized. Current active State will be paused and removed from stage, if any.

Parameters

state:State

addInactiveState()method 
public function addInactiveState(state:State):void

Add an inactive State. Add a State without setting it to be active. This will add the State to the bottom of the State stack without initialization.

Parameters

state:State

deactivateActiveState()method 
public function deactivateActiveState(deInit:Boolean = false):void

Deactivate current active State. This will pause the current active State and move it to the bottom of the stack. The current top State will be set as the active State.

Parameters

deInit:Boolean (default = false)

removeActiveState()method 
public function removeActiveState():void

Remove current active State. This will pause, deinitialize and remove the current active State from the stack. It will be clean up when handleCleanUp() is called. The current top State will be set as the active State.

Reset current active State. This will call reset() of the current active State.
